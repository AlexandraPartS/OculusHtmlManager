Панель управления VSOC

Добро пожаловать! Здесь Вы можете обновить конфигурацию VSOC.
Для продолжения работы выберите раздел:

  1. Чтобы посмотреть пароль, кликните по иконке рядом с ним

  2. Чтобы скрыть пароль, кликните по иконке рядом с ним ещё раз

  3. После изменения паролей перенесите data.json на Ваши Oculus Quest 2 в соответствии с инструкцией из коплекта.

  4. ВАЖНО: Тщательно проверяйте пароли на правильность перед отправкой, иначе Ваша учетная запись может быть заблокирована. В таком случае необходимо обратиться в тех. поддержку.

Изменить пароли Отменить

  1. Вы можете изменить IP-адрес сервера в рамках подсети 172.16.0.0/32, если он не занят.
    При смене IP-адреса сервисы автоматически перезапускаются.

  2. Также Вы можете посмотреть здесь статус ключевых сервисов VSOC. Если хотя бы один из сервисов выключен, запуск комплекса невозможен.

Если все пароли правильные, а сервисы — запущены, осталось настроить VR-комплекты в соответствии с инструкцией.

Если у Вас возникли вопросы — свяжитесь с командой технической поддержки.
